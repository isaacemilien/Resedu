Week 1: Introduction to Web Development

An opener to the basics of web development

By Isaac Emilien

October 8, 2022

This week's lectures covered the fundamentals of web development. This covers responsive web design, standards, and reasoning behind HTML and CSS rules. Further knowledge gained about these were:

Responsive Design

Responsive design is the adjusting of website scaling based on the device screen. Meaning that the content can be seen on a large variety of devices.

While learning this topic there was difficulty understanding how basic responsive design worked, from reading [1] interaction-design.org’s article and new understanding that responsive design is implemented based first on by reading the width of a user's device to gauge how to size elements.

Web Standards

Web standards are a set of rules for displaying information on the web. Web standards allow for information to be shown in a similar way on different platforms.
